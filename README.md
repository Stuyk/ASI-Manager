# ASI-Manager
An .ASI Manager for GTA: San Andreas <br />
First program I ever wrote in C#.

### Installation
Download <br />
Place in your Grand Theft Auto San Andreas Directory <br />
Run as Administrator <br />
That's it! <br />

### Requirements:
- .Net 4.5+ <br />
- Silent's .ASI Loader <br />
